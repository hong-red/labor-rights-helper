# 云服务器部署指南 - 维权助手统计系统

## 1. 服务器准备

### 系统要求
- **操作系统**: Ubuntu 20.04/22.04 LTS 或 CentOS 7/8
- **内存**: 最少512MB，推荐1GB+
- **存储**: 最少2GB可用空间
- **网络**: 开放端口 80/443/3000

### 安全组配置
```
入站规则：
- TCP 80 (HTTP)
- TCP 443 (HTTPS) 
- TCP 3000 (应用端口)
- TCP 22 (SSH)
```

## 5. Nginx配置

### 创建站点配置
```bash
sudo nano /etc/nginx/sites-available/weiquan
```

### 配置文件内容
```nginx
server {
    listen 80;
    server_name your-domain.com www.your-domain.com;

    # 前端静态文件
    location / {
        root /home/user/;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 后端API代理
    location /api/ {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }

    # 安全头设置
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;
    add_header Referrer-Policy "no-referrer-when-downgrade" always;
    add_header Content-Security-Policy "default-src 'self' http: https: data: blob: 'unsafe-inline'" always;
}
```

### 启用站点
```bash
sudo ln -s /etc/nginx/sites-available/weiquan /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

## 6. SSL证书配置 (HTTPS)

### 使用Let's Encrypt
```bash
# 安装Certbot
sudo apt install certbot python3-certbot-nginx  # Ubuntu/Debian
sudo yum install certbot python3-certbot-nginx  # CentOS/RHEL

# 获取证书
sudo certbot --nginx -d your-domain.com -d www.your-domain.com

# 自动续期
sudo crontab -e
# 添加: 0 12 * * * /usr/bin/certbot renew --quiet
```

## 7. 数据库管理

### 备份数据库
```bash
# 创建备份脚本
sudo nano /home/user/backup-db.sh
```

```bash
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="/home/user/backups"
DB_PATH="/home/user/backend/stats.db"

mkdir -p $BACKUP_DIR
cp $DB_PATH $BACKUP_DIR/stats_backup_$DATE.db
echo "数据库备份完成: stats_backup_$DATE.db"
```

```bash
chmod +x /home/user/backup-db.sh
# 设置定时备份
crontab -e
# 添加: 0 2 * * * /home/user/backup-db.sh
```

## 9. 故障排查

### 常见问题

1. **端口被占用**
```bash
sudo lsof -i :3000
sudo kill -9 PID
```

2. **权限问题**
```bash
sudo chown -R user:user /home/user/
sudo chmod -R 755 /home/user/
```

3. **Nginx配置错误**
```bash
sudo nginx -t
sudo systemctl status nginx
sudo journalctl -u nginx -f
```

4. **数据库锁定**
```bash
# 停止应用
pm2 stop weiquan-stats

# 检查数据库
sqlite3 stats.db "PRAGMA integrity_check;"

# 重启应用
pm2 start weiquan-stats
```

## 11. 性能优化

### 数据库优化
```bash
# 定期清理旧数据（可选）
sqlite3 stats.db "DELETE FROM feature_clicks WHERE last_clicked < datetime('now', '-90 days');"

# 重建索引
sqlite3 stats.db "REINDEX;"
```

### 系统优化
```bash
# 增加文件描述符限制
echo "* soft nofile 65536" | sudo tee -a /etc/security/limits.conf
echo "* hard nofile 65536" | sudo tee -a /etc/security/limits.conf

# 优化内核参数
echo "net.core.somaxconn = 65536" | sudo tee -a /etc/sysctl.conf
sudo sysctl -p
```
